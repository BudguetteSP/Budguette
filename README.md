# Budguette  

**Budguette** is a budgeting app designed to help users manage expenses, set budgets, track subscriptions, and visualize savings with a clean and modern interface.  

👉 **Full project documentation is available here:**  
[📖 Budguette-Milestones Repository](https://github.com/BudguetteSP/Budguette-Milestones)  

---

## Features  
- 💰 Expense and budget tracking  
- 📅 Subscription management  
- 📊 Savings visualization  
- 🔐 Firebase authentication (Google, Facebook, Email)  
- 🔄 Real-time syncing across devices
